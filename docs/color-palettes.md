# Grentu Code — Цветовые палитры

## Вариант A: "Neon Forest" 🌲
Изумрудно-зелёный с голубыми акцентами. Технологичный, свежий.

```
Primary:    #00FF87  (яркий мятный зелёный)
Secondary:  #60EFFD  (электрик-голубой)
Accent:     #FA709A  (розовый для ошибок/warnings)
Text:       #E0E0E0  (светло-серый)
Muted:      #6B7280  (тёмно-серый)
Bg:         #0D1117  (почти чёрный, как GitHub dark)
```
Вайб: современный, технологичный, "зелёный терминал будущего"

---

## Вариант B: "Sunset Terminal" 🌅
Тёплый градиент оранжевый → розовый. Киберпанк-эстетика.

```
Primary:    #FF8C42  (тёплый оранжевый)
Secondary:  #FF3D7F  (ярко-розовый)
Accent:     #FFD23F  (золотой для highlights)
Text:       #F5F5F5  (белый)
Muted:      #8B7355  (тёплый серый)
Bg:         #1A0F0A  (тёмно-коричневый)
```
Вайб: тёплый, энергичный, выделяется среди конкурентов

---

## Вариант C: "Deep Space" 🌌
Синий + фиолетовый. Космическая эстетика, спокойный.

```
Primary:    #7C3AED  (фиолетовый)
Secondary:  #3B82F6  (синий)
Accent:     #22D3EE  (циан для ошибок/warnings)
Text:       #E2E8F0  (светло-серо-голубой)
Muted:      #475569  (тёмно-синий серый)
Bg:         #0F0F1E  (очень тёмный фиолетовый)
```
Вайб: премиум, спокойный, "профессиональный инструмент"

---

## Вариант D: "Hacker Green" 💚
Классический терминальный зелёный, минималистичный.

```
Primary:    #39FF14  (неоновый зелёный)
Secondary:  #00D9FF  (циан)
Accent:     #FF6B6B  (красный для ошибок)
Text:       #C8FFC8  (мягкий зелёно-белый)
Muted:      #4A7C4A  (тёмно-зелёный)
Bg:         #0A0A0A  (чистый чёрный)
```
Вайб: хакерский, ностальгический, "Matrix meets modern CLI"

---

## Рекомендация

Для AI-ассистента кодинга в 2025 лучше всего подходит **A (Neon Forest)** или **C (Deep Space)**:
- **A** — свежий, запоминающийся, зелёный = "код работает"
- **C** — премиум-чувство, как Linear/Vercel

**B** — самый уникальный, но может утомлять глаза при долгой работе.
**D** — ностальгический, но может выглядеть "дёшево".

## Соответствие ролей цветов

| Роль | Назначение |
|------|-----------|
| Primary | Баннер, prompt-символ `❯`, заголовки |
| Secondary | Имя ассистента, ссылки, tech-детали |
| Accent | Ошибки, warnings, важные действия |
| Text | Основной текст ответов |
| Muted | Метаданные, timestamps, подсказки |
| Bg | Фон (если поддерживается терминалом) |
