## Tagline

```
  AI coding agent · v0.1.0
```

## Цвета баннера (по умолчанию)

| Элемент | Цвет |
|---------|------|
| Сам баннер (буквы) | Cyan, Bold (`\033[96m\033[1m`) |
| Tagline | Green (`\033[92m`) |
| Версия | Dim (`\033[2m`) |

## Пример рендера


## Параметры

- **Фиксированный размер** — не зависит от ширины терминала
- **Ширина баннера:** 56 символов
- **Высота:** 5 строк + 1 пустая сверху + 2 строки tagline + 1 пустая снизу
- **Цвета применяются через выбранную тему** (при onboarding можно выбрать палитру, а цвет баннера берётся из неё)
